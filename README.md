<h1>Gestão de Frotas</h1>
Desenvolvi um Aplicaivo Web com ASP.NET e SQL Server como SGBD. Com este Aplicativo é possível <i>cadastrar, editar, deletar e visualizar</i> dados sobre: <strong>Veículos, Motoristas, Viagens e Manutenções</strong>. Também foi usado bootstrap para estilos. O site pode ser encontrado live pela url: http://gestaodefrotas.somee.com.
